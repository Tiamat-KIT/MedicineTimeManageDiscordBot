# MedicineTimeManageDiscordBot
薬を飲む時間を管理できるDiscordBotを開発します

1. `yarn init`
2. `yarn add discord.js dotenv`
3. `yarn add -D typescript ts-node eslint`
4. `yarn tsc --init`
5. `yarn eslint --init`

## 使用するものについて
- Discord.js

- TypeScript

- [Volta](https://volta.sh)
    1. Install Volta
    `curl https://get.volta.sh | bash`
    2. Node Install in Volta
    `volta install node`
    3. Install Yarn
    `volta install yarn`
